    # 03 样式继承与其他概念

    # 1. 继承

    样式的继承，为一个元素设置的样式，同时也会应用到它的后代元素上。继承是发生在祖先与后代之间的。

    • 继承的设计是为了方便开发
      • 利用继承可以将一些通用的样式，统一设置到共同的祖先上。这样只需要设置一次即可让所有的元素都具有该样式
      • 并不是所有的样式都会被继承。比如背景相关，布局相关等样式都不会被继承。

    # 2.选择器的权重

    当通过不同的选择器，选中相同的元素，并且为相同的样式设置不同的值时，此时就发生了样式的冲突，样式冲突时，样式由选择器的权重（优先级）决定选择器的权重。

    选择器 权重
    行内样式 1，0，0，0
    ID选择器 0，1，0，0
    类和伪类选择器 0，0，1，0
    元素选择器 0，0，0，1
    通配选择器 * 0，0，0，0
    继承的样式 没有优先级

    比较优先级时，需要将所有的选择器的优先级进行相加计算，最后优先级越高，则越优先显示（分组选择器时单独计算的）

    • 选择器的累加不会超过其最大的数量级，类选择器再高也不会超过ID选择器
    • 如果优先级计算后相同，则优先使用靠下的样式
    • 可以再某一个样式后添加!important，此时该样式会获取到最高的优先级，甚至超过内联样式，注意：再开发中一定要慎用。
    <style>
    #box1 {
        background-color: orange;
    }
    div#box1 {
        background-color: yellowgreen;
    }
    .red {
        background_color: red;
    }
    div, p, span {
        backgorund-color: aqua;
    }
</style>

<div id="box1" class="red">
    选择器的权重

    # 3. 长度单位

    像素： 显示器屏幕实际是由一个个的小点（单位色块，即像素）构成的

    1. 像素和分辨率的关系

    ​ 分辨率 = 水平方向像素 * 垂直方向像素

    • 不同屏幕的像素大小不同，像素越小，屏幕显示效果越清晰。所以同样的像素大小在不同的设备上显示的效果是不同的。
    • 同一台设备的像素大小是不变的，把图片放大超过100%时占的像素点多，但是图像也会变模糊。

    ​ 2. 屏幕实现图片放大和缩小的原理

    • 其实是设备通过算法对图像进行了像素补足。同理，把图片按小于100%缩放时，也是通过算法将图片元素减少。

    百分比：也可以将属性设置为相对于其父元素属性的百分比

    • 使子元素跟随父元素的改变而改变

    em： em是相对于元素的字体大小来计算的，1em = <self>.font-size * 10

    • 也就是说，em值会根据元素本身的字体大小改变而改变，浏览器默认的font-size为16px

    rem： rem是相对于根元素的字体大小来计算的, 1em = <root>.font-size * 10

    • 也就是说 rem的值会根据根元素的字体大小改变而改变

      <style>
    * {
        font-size: 24px;
    }
    
    .box1 {
        width: 200px;
        height: 200px;
        background-color: orange;
    }
    
    .box2 {
        width: 50%;
        height:50%;
        background-color: aqua;
    }
    
    .box3 {
        font-size: 20px;
        width:10em;
        height:10em;
        background-color: greenyellow;
    }
    .box4 {
        font-size: 20px;
        /*
        当用rem时，不管怎么改变本元素的font-size都不会变。
        需要定义root元素的font-size才可以
        */
        width: 10rem;
        height: 10rem;
        background-color: red;           
    }
</style>

    <div class="box2">

    # 4. 颜色单位

    css中的颜色名称

    • 在css中可以直接使用颜色名来设置颜色，比如：red， orange， yellow， blue等
    • 其中有140多个颜色名称时所有浏览器都支持的，但是有个问题，在css中直接使用颜色名称非常不方便。

    # RGB值

    RGB通过三原色的不同浓度来调配出不同的颜色

    • 语法：RGB(red, green, blue)
    • 范围：每一种颜色的范围在0~255（0%~100%）之间

    # RGBA

    在RGB的基础上增加了一个a表示不透明度

    • 语法：RGBA(red, green, blue, .5)
    • 1表示完全不透明
    • 0表示完全透明
    • 0.5表示半透明

    # 十六进制的RGB值

    即RGB值的十六进制写法

    • 语法：#RRGGBB
    • 范围：每种颜色的范围在00~ff之间
    • 如果颜色两位重复可以进行简写，如 #AABBCC => #ABC
      • 在vscode中，我们可以看到其对于颜色进行预览展示，并且将鼠标移至color处悬停，会弹出rgb调色板，方便我们进行调色。

    # HSL

    • 语法： HSL(色相， 饱和度， 亮度)
    • H 色相 0~360
    • S 饱和度 0%~100%
    • L亮度 0%~100%
      • 取色器FastStone Capture，下载地址：FastStone Capture - Download (opens new window)
